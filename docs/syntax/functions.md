# Functions

Functions are used in [expressions](/syntax/#expression). They can take arguments and return values.

## Built-in Functions


Here is the list of the builtin functions which can be used in [expressions](#expressions)

### String

- `split`
- `lower`
- `upper`
- `title`
- `join`
- `trim`
- `trimLeft`
- `trimRigh`
- `trimPrefix`
- `trimSuffix`
- `replace`
- `replaceAll`

### Math

- `round`
- `ceil`
- `abs`
- `floor`
- `max`
- `min`

### Date & Time

- `date`
- `now`
- `year`
- `month`
- `weekday`
- `day`
- `hour`
- `minute`
- `second`

### Type Conversion

- `bool`

### Helpers

- `ternary`
